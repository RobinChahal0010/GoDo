# GoDo 📝

A simple and fast CLI-based To-Do application built with Go.

---

## ✨ Features

- Add tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Simple terminal interface
- Lightweight and beginner-friendly

---

## ⚙️ Tech Stack

- Go (Golang)
- Standard Library Only

---

## 📁 Project Structure

```bash
GoDo/
│── main.go
│── README.md
│── .gitignore
```

---

## 🚀 Run Locally

```bash
git clone https://github.com/RobinChahal0010/GoDo.git

cd GoDo

go run main.go
```
